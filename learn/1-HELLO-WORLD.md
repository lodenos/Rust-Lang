# Hello World!
creat a file with extention **.rs**
```
touch main.rs
```
Write the hello world code
```rust
fn main() {
  println!("Hello, world!");
}
```
Compile with **rustc**
```
rustc main.rs
```
Run
```
./main
```
Output
```
Hello, world!
```
source : https://doc.rust-lang.org/book/ch01-02-hello-world.html
